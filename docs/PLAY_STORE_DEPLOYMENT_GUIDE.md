## Step-by-Step (End-to-End)

### 1) Confirm Release Metadata (Done)
- `versionName` = 2.7.1
- `versionCode` = 39
- Changelog already includes 2.7.1

### 2) Run Tests (Required Before Release)
**Unit tests**
```bash
./android/gradlew -p android :app:testDebugUnitTest
```

**Instrumentation/UI tests (emulator required)**
```bash
./android/gradlew -p android :app:connectedDebugAndroidTest
```

**Screenshot verification (required):**
Look for screenshots at:
```
android/app/build/reports/androidTests/connected/debug/screenshots/AndroidTestScreenshots/
```
Confirm they exist and match expected UI.

### 3) Build the Play Store Bundle (AAB)
```bash
./android/gradlew -p android :app:bundleRelease
```
Output:
```
android/app/build/outputs/bundle/release/app-release.aab
```

### 4) (Optional) Build Release APK for Local Validation
Only after commit/push, per release discipline:
```bash
./android/gradlew -p android :app:assembleRelease
```
Rename to:
```
cosmic-ocean-v2.7.1.apk
```

### 5) Play Console Setup (You in Console)
- Create/confirm app entry
- Upload `app-release.aab`
- Fill **Data Safety** (use `docs/DATA_SAFETY_DRAFT.md` as a starting point)
- Add **Privacy Policy URL** (host `docs/PRIVACY_POLICY.md` somewhere public)
- Fill **Store Listing** (use `docs/STORE_LISTING_COPY.md`)
- Upload listing assets (see notes in listing copy doc)
- Submit for review
