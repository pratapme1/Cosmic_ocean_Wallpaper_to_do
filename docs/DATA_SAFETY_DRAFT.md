# Play Console Data Safety Draft — Cosmic Ocean (Local-Only)

**Important:** Use this as a starting point only. In Play Console, answer based on the current build’s behavior.

## Summary (Expected for Local-Only)
- **Data collected:** None (no data transmitted off-device)
- **Data shared:** None
- **Data processed on device only**
- **No account or login required**

## Data Types (If Asked)
If Play Console asks about data types, the app stores the following **locally**:
- Tasks and task metadata
- App settings and preferences
- Wallpaper and display settings

## Purpose (If Asked)
All local data is used only to provide core app functionality.

## Notes
- No backend sync or account system in 2.7.1
- No third-party analytics or crash reporting SDKs
- Data does not leave the device
